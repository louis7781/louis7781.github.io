产品列表

Splunk Enterprise

Splunk Enterprise是从机器数据中汇总、分析和获取答案的最佳捷径。数据分析以询问一切事情。

Splunk IT 服务智能 (ITSI)

使用针对IT环境所量身定制的威胁监控和数据分析解决方案，简化操作，确定事件处理的优先级，并使 IT 与业务保持一致。

Splunk Enterprise Security

通过可操作的情报获得对安全状况的端到端可视性监控，帮助您确定事件处理优先级顺序并快速采取行动。

Splunk 用户行为分析

通过机器学习提供支持的用户和实体行为数据进行分析，防范内部威胁。

Phantom

使用 Phantom 安全运营平台，您可以充分利用平台现有的资源进行企业运营环境的安全监控。

Splunk Enterprise

使用指标加快查询速度

利用指标 - 数字和时间序列数据 - 来提升分析企业数据的性能。轻松将事件日志数据转换为指标，并提供可视化分析结果以及显著增强监视和警报功能。

Splunk Enterprise Security

提高企业信息安全操作

使用警报管理、风险评分以及可自定义的仪表板和可视化来识别、确定事件处理优先级并管理企业网络安全事件。

Splunk IT Service Intelligence(ITSI)

服务分析器树

使用机器学习来获取正确的数据并进行大数据智能分析，以此快速确定事件的服务或应用程序来源，了解根本原因。

Splunk 用户行为分析

增强可见性和检测

使用机器学习自动进行用户行为分析和威胁检测，这样您就可以花更多的时间使用基于行为分析的警报来快速查看和解决问题。

Phantom

自动化

Phantom 使您能够在几秒钟内在安全基础架构内执行操作，从而更智能地工作，而如果手动执行则需要数小时或更长时间。使用我们的可视化编辑器（无需编码）或集成的 Python 开发环境，将您的工作流程编码为自动化的剧本。
